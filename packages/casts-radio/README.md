# @casts/radio

> a input react component

[![NPM](https://img.shields.io/npm/v/@casts/radio.svg)](https://www.npmjs.com/package/@casts/radio) [![JavaScript Style Guide](https://img.shields.io/badge/code_style-standard-brightgreen.svg)](https://standardjs.com)

## Install

```bash
npm install --save @casts/radio
```

## Usage

```tsx
import React, { Component } from 'react';

import MyComponent from '@casts/radio';
import '@casts/radio/dist/cjs/index.css';

class Example extends Component {
  render() {
    return <MyComponent />;
  }
}
```
