# @casts/icons

> a input react component

[![NPM](https://img.shields.io/npm/v/@casts/icons.svg)](https://www.npmjs.com/package/@casts/icons) [![JavaScript Style Guide](https://img.shields.io/badge/code_style-standard-brightgreen.svg)](https://standardjs.com)

## Install

```bash
npm install --save @casts/icons
```

## Usage

```tsx
import React from 'react';
import { Home2Line } from '@casts/icons';
import '@casts/icons/dist/cjs/index.css';

export default () => <Home2Line />;
```
